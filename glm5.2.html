Comment ça marche ?
  1. Renseignez vos clés API dans « Paramètres API » (Mistral, OpenAI, Anthropic, Google…).
  2. Manuel : Cliquez « Ajouter un agent », choisissez un provider + modèle, rédigez le system prompt.
  3. Automatique : « Générer les agents IA » crée 7 spécialistes via mistral-large-2512. Vous pouvez ensuite les éditer pour changer leur provider/modèle.
  4. Chaque agent utilise sa propre clé API et son modèle au moment du pipeline.
  5. « Lancer la production » exécute le pipeline séquentiel ; chaque agent enrichit le JSON de production.
  6. « Pousser les scènes vers le storyboard » crée une carte par scène dans la timeline.
