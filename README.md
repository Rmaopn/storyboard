# 🎬 AI Cinema Studio v1

> **Framework de production cinématographique automatisé utilisant l'IA générative**

## 📖 Présentation

AI Cinema Studio v1 est un système complet de production vidéo qui transforme une simple idée en prompts techniques prêts pour Midjourney, Runway, Pika et ElevenLabs.

Conçu pour les créateurs de contenu qui veulent produire des vidéos de qualité professionnelle sans équipe de production.

## 🎯 Fonctionnalités

### 🧠 Orchestration Multi-Agents
- **7 agents spécialisés** : Showrunner, Architecte narratif, Créateur de personnages, etc.
- **Pipeline séquentiel** : Chaque agent enrichit la production précédente
- **Multi-provider** : Mistral, OpenAI, Anthropic, Google AI
